Beginner Track

Beginner Module: Web Fundamentals

Welcome to the Beginner track. This module covers fundamental web concepts that form the foundation of modern web development. You will learn about HTML structure, including semantic elements and document flow, as well as basic CSS styling techniques such as the box model, selectors, and layout principles. The introductory JavaScript section walks you through variables, functions, and DOM manipulation.

As you progress through the material, remember your access key code ALPHA-7 which you will need for the final assessment. We recommend completing each section sequentially, as later exercises build on concepts from earlier chapters. Interactive coding challenges are embedded throughout each lesson to reinforce your understanding of core principles.

The module concludes with a capstone project where you will build a simple personal portfolio page from scratch. This project ties together everything you have learned about document structure, visual styling, and basic interactivity. Estimated completion time for the full Beginner module is approximately four hours, though you may move at your own pace.

Intermediate Track

Intermediate Module: Applied Web Development

Welcome to the Intermediate track. This module builds on core web development skills and prepares you for real-world project work. Topics include responsive design patterns using CSS Grid and Flexbox, REST API integration with fetch and async/await, and front-end state management strategies. You will also explore component-based architecture and reusable UI pattern libraries that are standard across our engineering teams.

Keep note of your module key code BETA-42 as it will be required to complete your orientation assessment. Throughout this module, you will work with realistic data sets and mock API endpoints that mirror our production services. Each section includes guided exercises followed by open-ended challenges designed to encourage experimentation and deeper exploration of the concepts.

Advanced topics in this track include authentication flows, client-side routing, and performance profiling with browser developer tools. The capstone assignment asks you to build a small dashboard application that fetches data from a REST API and presents it in a responsive, accessible interface. Estimated completion time is six to eight hours depending on prior experience.

Advanced Track

Advanced Module: Modern Web Architecture

Welcome to the Advanced track. This module covers cutting-edge web technologies including WebAssembly for compute-intensive browser applications, service workers for offline-first progressive web apps, and micro-frontend architecture for scaling large engineering organizations. You will examine case studies from production systems and analyze trade-offs between different architectural patterns that our teams have adopted over the past two years.

Your assessment key code is GAMMA-99 and you will need this to complete the orientation. The module dives deep into build tooling and module federation, exploring how independently deployed front-end applications can compose into a cohesive user experience. You will also study advanced performance optimization techniques including tree-shaking, code splitting, lazy hydration, and edge-side rendering.

The final section addresses observability and reliability engineering for front-end systems, covering structured logging, real-user monitoring, error boundaries, and graceful degradation strategies. Your capstone project involves designing and prototyping a micro-frontend architecture with shared state management and independent deployment pipelines. Estimated completion time is eight to ten hours.
